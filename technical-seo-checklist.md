# Technical SEO Checklist

A practical technical SEO checklist for websites, agencies and freelancers.

## Indexation

- Check if important pages are indexable
- Review robots.txt rules
- Check meta robots tags
- Review canonical tags
- Check XML sitemap availability
- Submit sitemap in Google Search Console

## Crawlability

- Avoid blocking important pages
- Fix broken internal links
- Check redirect chains
- Keep important pages within a reasonable click depth
- Use clean internal linking

## On-page basics

- Use one clear H1 per page
- Write unique title tags
- Write useful meta descriptions
- Use descriptive headings
- Add image alt text where appropriate

## Structured data

- Add Organization schema
- Add LocalBusiness schema for local companies
- Add FAQ schema where suitable
- Validate structured data before publishing

## Performance

- Compress images
- Use modern image formats
- Minimize unused CSS and JavaScript
- Check Core Web Vitals
- Improve server response time

## Maintenance

- Review Google Search Console regularly
- Monitor 404 errors
- Update outdated content
- Check redirects after website changes
- Review sitemap and robots.txt after launches
