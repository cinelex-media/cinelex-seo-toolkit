# Cinelex SEO Toolkit

Open-source technical SEO toolkit for agencies, freelancers, website owners and developers.

This repository contains practical SEO resources for auditing, improving and maintaining websites. It focuses on technical SEO, local SEO, structured data, redirects, robots.txt examples, performance checks and repeatable audit workflows.

## What is included

- Technical SEO checklist
- Local SEO checklist
- Structured data snippets
- Robots.txt examples
- Redirect examples
- SEO audit template
- Website maintenance workflows

## Who is this for?

This toolkit is useful for:

- Webdesign agencies
- SEO consultants
- Freelancers
- WordPress developers
- Local businesses
- Website owners

## Why this exists

Many small businesses and agencies need practical, understandable SEO workflows without relying on expensive proprietary tools for every basic check. This project collects reusable templates, snippets and checklists that can be improved by the community.
