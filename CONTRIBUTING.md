# Contributing

Thanks for your interest in contributing to Cinelex SEO Toolkit.

This project is focused on practical, reusable SEO resources for agencies, freelancers, developers and website owners.

## Ways to contribute

You can help by:

- Improving technical SEO checklists
- Adding structured data examples
- Adding robots.txt examples
- Adding redirect examples
- Improving audit templates
- Reporting outdated recommendations
- Adding CMS-specific SEO workflows

## Guidelines

Please keep contributions:

- Practical
- Clear
- Easy to understand
- Useful for real websites
- Based on current SEO best practices

## Suggested format

When adding a checklist or guide, use clear sections, short explanations and actionable steps.
